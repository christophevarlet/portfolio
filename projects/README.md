# Projects

Ce dossier contient les projets techniques et prototypes.  

Chaque projet devrait inclure :
- **Contexte et objectif** : pourquoi le projet a été réalisé
- **Technologies utilisées** : langages, frameworks, outils
- **Organisation des fichiers** : structure du code, notebooks, assets
- **Instructions** : comment exécuter ou tester le projet

Les projets sont documentés pour montrer **la démarche, l’expérimentation et les résultats**.  
Chaque sous-dossier de projet contient son propre README détaillé.

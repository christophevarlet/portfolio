# Explorations

Ce dossier regroupe les journaux exploratoires et expérimentations.  

Sous-dossiers possibles :
- `data/` : analyses de données, visualisations, insights
- `nlp/` : expérimentations en traitement du langage naturel
- `automation/` : prototypes et réflexions sur l’automatisation de tâches

Chaque fichier Markdown doit inclure :
- Contexte et objectif de l’exploration
- Hypothèses ou questions de départ
- Démarche et expérimentations
- Résultats et observations
- Synthèse ou réflexions complémentaires

> Chaque exploration est un petit **journal méthodologique**, permettant de suivre la progression et la pensée derrière les résultats.
